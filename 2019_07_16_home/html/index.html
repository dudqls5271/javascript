아두이노 버튼을 이용한 피에조 부저 제어 (예제)

글쓴이 : Lsoleix ・ 2019. 1. 9. 21:09 ・ 비공개

피에조 부저란?
피에조 부저는 피에조를 이용하여 소리를 내는 작은 스피커이다. 피에조는 특정 방향으로 압력을 가하면 결정체(수정, 전기석 등)의 표면에서 전기가 발생하는 성질을 이용한 것으로 여기에 얇은 판을 붙여 미세한 떨림으로부터 소리가 나게 된다. 소리가 크지 않다는 단점이 있지만 세밀하게 조작하면 음악도 연주할 수 있다. 피에조 부저는 극성(+, -)가 있다. 부저 윗면에 +라고 써져 있거나 조그만 홈이 파여있는 쪽에 +극을 연결하면 된다.

소프트웨어 Coding

 
                        int buzzer = 7;
                        int button = 6;
                        int btnState = 0;
                                                       
                        void setup() {
                        pinMode(buzzer, OUTPUT);
                        pinMode(button, INPUT);
                        }
                                                       
                        void loop() {
                        btnState = digitalRead(button);
                                                       
                        if(btnState == 1)
                        {
                        digitalWrite(buzzer, HIGH);   
                        }
                        else{
                        digitalWrite(buzzer, LOW);
                        }
                        delay(10); 
                        }

위의 소스코드를 보면 버튼으로 LED 제어했던 코드랑 매우 비슷하다는 것을 느낄 수가 있다.
간단하게 해석을 해보면

if(btnState == 1) 만약 버튼이 눌리면

digitalWrite(buzzer, HIGH); 부저를 켜라

else 아니면

digitalWrite(buzzer, LOW); 꺼라


​
위와 같은 식으로 해석이 가능하다.

아두이노 가변저항을 이용한 서보모터 제어

글쓴이 : Lsoleix ・ 2018. 12. 26. 12:00
빨간색 저번에 서보모터를 올린 적이 있다. 오늘은 이걸을 저번에 올렸던 가변저항을 이용하여 서보모터를 돌려올 것이다. 아래의 링크에는 서보모터와 가변저항의 예제들이 있다.

서보모터: http://lsoleix.tk/221425927122

가변저항: http://lsoleix.tk/221414605812

소프트웨어 Coding

 
                        #include 

                        ervo myservo;
                                    
                        void setup() {
                         myservo.attach(13);
                        }
                                    
                        void loop() {
                        int val = analogRead(A0);
                        int radian = map(val, 0, 1023, 0, 120);
                        myservo.write(radian);
                        delay(20);
                        }
                                        
                                
 
 이제 loop 부분을 설명하자면 우선

                        ​

int val = analogRead(A0); ---->아날로그 0번 핀의 값을 val에 저장한다.

int radian = map(val, 0, 1023, 0, 120); -----> 입력값을 목표 값 으로 변경함.

 myservo.write(radian); -----> 변경된 만큼 서보모터를 움직이게 함

delay(20); ----> 0.02초 지연

간단하게 설명하면 가변저항의 값을 저장해서 그 값만큼 서보모터를 돌린다고 생각하면 된다.

오늘은 가변저항으로 서보모터를 제어해봤는데 서보모터, 가변저항 예제를 잘 보고 왔다면 잘 이해할 수 있을 것이다.

아두이노 LED_7(RGB LED)

글쓴이 : Lsoleix ・ 2018. 12. 25. 12:00

RGB란?
RGB 가산 혼합(RGB)은 빛의 삼원색을 이용하여 색을 표현하는 방식이다. R는 빨간색, G는 초록색, B는 파란색을 의미한다. 이 3개의 색을 섞어서 총 4가지의 색을 나타낼 수 있다.

RGB LED란?

 
 한쪽 방향으로 전류가 흐르도록 제어하는 반도체 소자를 다이오드라 한다. 

 다이오드 중에 전기 에너지를 빛 에너지로 변환하는 것을 발광 다이오드, 즉, LED (Light Emitting Diode)라고 한다. 
 3색 LED는 다양한 색상의 LED에서 빨간색, 초록색, 파란색 3개의 LED를 하나의 LED로 합친 LED를 3색 LED라고 한다.
 빛의 3원 색인 빨강, 초록, 파란색 3가지 빛을 합성을 이용하여 다양한 색을 표현한다.

사용방법

 
위의 사진과 같이 위쪽부터 빨간색 초록색 파란색으로 이루어져 있다. 그리고 가장 긴 핀은 GMD이다. 
일반 LED와 반대로 긴 단지에 -전극을, 나머지 짧은 단자에 + 전극을 연결해야 LED가 켜진다.

색의 조합
색 색 색
빨간색 초록색 노란색
빨강색 파란색 보라색
파란색 초록색 청록색
빨강색 초록색 파란색 하얀색

소프트웨어 Coding

 
                            int R = 11;
                            int G = 10;
                            int B = 9;
                            
                            
                            void setup() {
                            
                              pinMode (11,OUTPUT);
                              pinMode (10,OUTPUT);
                              pinMode (9,OUTPUT);
                              
                            }
                            
                             
                             
                            void loop() { 
                            
                              digitalWrite (11,HIGH);  //빨강색이 켜짐
                              digitalWrite (10,LOW);
                              digitalWrite (9,LOW);
                              delay (1000);
                            
                              digitalWrite (11,LOW);
                              digitalWrite (10,HIGH);  //초록색이 켜짐
                              digitalWrite (9,LOW);
                              delay (1000);
                              
                              digitalWrite (11,LOW);
                              digitalWrite (10,LOW);
                              digitalWrite (9,HIGH);  //파란색이 켜짐
                              delay (1000);
                            
                              digitalWrite (11,HIGH);
                              digitalWrite (10,HIGH);  //하얀색이 켜짐
                              digitalWrite (9,HIGH);
                              delay (1000);
                            } 
                                                    
                                            
 
 위의 코드를 보면 지금까지 ELD1~6까지 매우 흡사하다. 굳이 이 코드를 설명하자면 

digitalWrite (11, HIGH); 11핀을 활성화시킨다는 것이다. HIGH 면 LED의 불이 켜지고 LOW 면 불이 꺼진다.
